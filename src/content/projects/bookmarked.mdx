## Overview

This project is a speech-based language classification system developed using the **RoDia** dataset. The classifier detects Romanian dialects from audio samples, covering five distinct regional dialects.

Built with Python and TensorFlow, the system demonstrates a full pipeline for preprocessing, training, and evaluating audio models in low-resource language settings.

## üß© Dataset Overview

The **RoDia** dataset focuses on dialect detection in Romanian speech and includes:

- **Ard**: Ardeal  
- **Ban**: Banat  
- **Mol**: Moldova  
- **Mun**: Muntenia  
- **Olt**: Oltenia  

### Structure

| Directory              | Labels                    |
|------------------------|---------------------------|
| `real_data/data/train` | Ard, Ban, Mol, Mun, Olt   |
| `real_data/data/test1` | Ard, Ban, Mol, Mun, Olt   |

Each label folder contains `.wav` audio files for that region's dialect.
